# HashCrack
hash cracking tool
### about :
HashCrack is a tool writed in python to perform hash cracking using rainbow table attack.
### installation : 
clone github repo
```
git clone https://github.com/o-sec/HashCrack.git
```

### usage :
```
usage : Hashcrack.py <hash>  <wordlist>
```
